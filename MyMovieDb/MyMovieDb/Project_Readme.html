Getting Started

How to get MyMovieRatings up and running.

This page relies on a SendGrid account to send emails for email account confirmation.
You'll need to tweak a couple things to get it up and running.

If you have SendGrid credentials

  • Create an AppSettingsSecrets.config file (or equivalent) containing an appSettings tag with "mailAccount" and "mailPassword" keys for your Sendgrid account and password.
  • In Web.config, replace file="..\..\..\AppSettingsSecrets.config" with an accurate path to your file.
  • Light 'em up. Everything should work just fine.

If you don't have SendGrid credentials, and aren't planning on getting any

  • Go to the Login HttpPost method in AccountController.cs
  • From lines 76-92, there is code to either validate emails (currently uncommented) and to proceed without a validated email (commented). Follow the commenting directions in the code. You'll uncomment a line and comment out a region.
  • Go to the Register HttpPost method in AccountController.cs.
  • From lines 177-190, there is more code to either validate email addresses (uncommented) and to proceed without validating the email address. Follow the commenting directions to uncomment two lines and comment out a region.

The connectionString in the Web.config is almost assuredly wrong

  • In the Web.config, change the DefaultConnection connection string to something accurate for your environment.
